# Tableau Custom Shapes - Pipeline Coverage Status

Use these PNG files for the Pipeline Coverage by Segment status marker.

Recommended:
- `pipeline-status-rounded-square-neutral.png` - neutral black rounded square. Use this first if Tableau tints the custom shape from the Color shelf.

Fallback colored versions:
- `pipeline-status-rounded-square-on-track.png` - teal, for On Track
- `pipeline-status-rounded-square-monitor.png` - amber, for Monitor
- `pipeline-status-rounded-square-at-risk.png` - red, for At Risk

Alternate:
- `pipeline-status-rounded-square-outline.png` - neutral outline version if the filled marker feels too heavy in the scorecard.

Tableau install path:
1. Copy the PNG files into `Documents\My Tableau Repository\Shapes\SaaS RevOps`.
2. In Tableau Desktop, open the worksheet.
3. On the Marks card, set mark type to Shape.
4. Click Shape, then More Shapes, then Reload Shapes.
5. Select the `SaaS RevOps` shape palette.
