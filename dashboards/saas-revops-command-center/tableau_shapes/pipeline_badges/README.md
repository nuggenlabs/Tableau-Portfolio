# Pipeline Coverage Badge Shapes

These PNGs recreate the rounded badge style shown in the reference image.

Recommended Tableau approach:
1. Use the blank badge PNGs as custom Shapes.
2. Put the dynamic coverage value and status on Label.
3. Center-align the label over the shape.

Use these blank shape files for Tableau:
- `coverage-badge-at-risk-blank.png`
- `coverage-badge-monitor-blank.png`
- `coverage-badge-on-track-blank.png`
- `coverage-badge-neutral-blank.png`

The sample files include static text and are for visual reference only:

- `coverage-badge-monitor-sample-2_30x.png`
- `coverage-badge-on-track-sample-4_70x.png`

- Status label: 6-7pt, bold, uppercase, muted blue-gray
- Alignment: center/middle
